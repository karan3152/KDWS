Instructions

Upload an Excel file (.xlsx or .xls) with the following columns:

  • name - Full name of the employee (will be split into first and last name)
  • email - Email address (must be unique)
  • phone_number - Contact phone number

The system will automatically:

  • Generate usernames based on email addresses
  • Create random secure passwords for each employee
  • Create employee accounts and profiles
